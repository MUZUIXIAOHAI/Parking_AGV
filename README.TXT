平台:
    正点原子mini STM32F103开发板
	
实现功能:
    同步左右轮子的方向，实现前后直行；
    实现停车AGV的抓取动作流程；
	
硬件资源:

    给定电机四路模拟量；（1*4）DA   ->转换成PWM
    接收四路编码器；（3*4）I/O
    给定电机电机方向信号四路，开关量；(1*4)I/O
    四个电机启停信号，开关量；(1*4)I/O
    天车八个开关量给定信号；(2*4)I/O


    （TIM2）PWM输出（5KHZ）：
    PA0、PA1、PA2、PA3；

    编码器输入：
    （TIM1）：PA8、PA9；-  右轮旋转编码器A、B相
    （TIM3）：PA6、PA7；-  左轮旋转编码器A、B相
    （TIM4）：PB6、PB7；-  左轮行走编码器A、B相；Z相接

    前后：
    左-M1：方向：PC11  启停：PC12
    右-M2：方向：PC1   启停：PD2

    左右：
    左-M1：方向：PC8   启停：PC9
    右-M2：方向：PC6   启停：PC7

    启停：控制要给低电平；
    方向：两边电机给定方向相反；
    
    
	
实验过程:
	板子上电，车辆向前行驶，检测到车辆前轮左右信号后，开始进行抓取动作。
	
注意事项:
	当前传感器就装了三个，上次去有一个没有带，三个也可以工作，但是四个工作比较稳定
